## 💻 How to use

```bash
git clone https://github.com/kangeunsong/Gangtic-Phone
cd repository
```

- Make sure that you're in right repository.

```bash
make
./server
./client nickname
```

- Do "make" first to make .exe files. Then, execute 1. server and 2. client(with your nickname).
